Iniciales en AppSheet usando ui-avatars.com

Este fragmento de código permite generar dinámicamente una imagen de avatar en AppSheet
basado en las iniciales del campo del nombre del usuario o de quien se desee, utilizando el servicio gratuito de https://ui-avatars.com.

Ideal para apps que requieren mostrar una imagen de usuario sin necesidad de subir fotos reales.

Tipo de campo en AppSheet: Virtual Column
Tipo de dato: Image

Fórmula:

CONCATENATE(
"https://ui-avatars.com/api/?name=",
ENCODEURL(LEFT([Nombre_Completo], 1)),
ENCODEURL(LEFT(INDEX(SPLIT([Nombre_Completo], " "), 2), 1)),
"&background=CCCCCC&color=000000&size=128"
)

Explicación del código:

"https://ui-avatars.com/api/?name=" → Base del servicio que genera el avatar.
ENCODEURL(LEFT([Nombre_Completo], 1)) → Extrae la primera letra del nombre completo.
ENCODEURL(LEFT(INDEX(SPLIT([Nombre_Completo], " "), 2), 1)) → Extrae la primera letra del segundo nombre o apellido.
"&background=CCCCCC&color=000000&size=128" → Personalización del avatar:
Fondo gris claro
Texto negro
Tamaño: 128px
Ejemplo:
Si [Nombre_Completo] = "César Pineda",
la imagen mostrará un avatar con las iniciales "C P".

Resultado URL:
"https://ui-avatars.com/api/?name=C+P&background=CCCCCC&color=000000&size=128"

Notas adicionales:

• Puedes cambiar los colores modificando los valores en los parámetros background y color.
• Si el nombre tiene más de dos palabras, puedes ajustar la fórmula para obtener el apellido o nombre deseado.
• La imagen se genera desde la nube, no ocupa espacio en tu app.
• Puedes usar esta URL directamente en otros campos tipo imagen si lo deseas.
